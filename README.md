# ConfigArc Launcher

Visual configuration and launcher for arcade titles using specific tool. Built with Tauri (Rust backend) and React + TypeScript (Vite) frontend.

## Prerequisites
- Node.js + npm
- Rust toolchain
- Tauri CLI

## Usage
```bash
npm install
npm run tauri dev
```

This starts the Vite dev server and the Tauri shell so you can edit configs, manage profiles, and launch games with selected profiles.
